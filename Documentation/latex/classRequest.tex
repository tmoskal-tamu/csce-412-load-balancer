\doxysubsection{Detailed Description}
Represents a request sent to the server. 

A \mbox{\hyperlink{classRequest}{Request}} contains information about the IP addresses involved, the amount of time required to process the request, and the type of job (e.\+g., \textquotesingle{}P\textquotesingle{} for processing, \textquotesingle{}S\textquotesingle{} for streaming).
