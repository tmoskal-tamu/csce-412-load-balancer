\doxysubsection{Detailed Description}
Manages a queue of requests to be processed by web servers. 

The \mbox{\hyperlink{classRequestQueue}{Request\+Queue}} class allows for adding requests to the queue and retrieving requests in a first-\/in-\/first-\/out (F\+I\+FO) order. It also checks whether the queue is empty.
