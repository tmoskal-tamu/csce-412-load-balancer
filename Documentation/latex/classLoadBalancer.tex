\doxysubsection{Detailed Description}
Manages the distribution of requests across multiple \mbox{\hyperlink{classWebServer}{Web\+Server}} instances. 

The \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} class generates requests, distributes them to web servers, and dynamically adds or removes servers based on the request load. It also tracks the total number of servers added/removed and the total number of requests created/completed.
